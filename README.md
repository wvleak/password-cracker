# Light Password Cracking Tool

Hash cracking tool built for learning purposes.

## Functionnalities

- Conduct a normal bruteforce attack with a wordlist
- Conduct a dictionary attack
- Convert a wordlist into a dictionary

## Hash Support

- md5
- sha1
- sha256
